# Game Developer API Utility Backlog

Goal: evolve this engine toward a Unity-like, game-developer-friendly API.

Approach: implement one class at a time, shipping small safe batches.

Current batch status:
- Transform: expanded in code in this batch.
- Other classes: backlog only (proposed).
